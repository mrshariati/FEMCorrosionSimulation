Create folder myTmpBuild inside the extracted folder petsc-3.13.0

sudo gedit(or nano) /etc/profile
Add the following lines to the end of file (note the configure command line):
PETSC_DIR=/home/reza/MySoftwares/petsc-3.13.0
PETSC_ARCH=DolfinCustomized18April
export PETSC_DIR PETSC_ARCH

reboot

./configure --with-packages-build-dir=/home/reza/MySoftwares/petsc-3.13.0/myTmpBuild --PETSC_ARCH=DolfinCustomized18April --download-hypre=yes --download-metis=yes --download-parmetis=yes --download-mumps=yes --download-scalapack=yes --download-suitesparse=yes

(in case of installation on previous build pass also --with-clean=1)

make all
make check

sudo ldconfig
